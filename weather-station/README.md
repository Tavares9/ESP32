# ESP32 Weather Station

## Overview
Embedded system developed using ESP32 for environmental data acquisition.

## Features
- Environmental data reading
- Sensor integration
- Serial communication output
- Real-time data processing

## Hardware
- ESP32
- DHT22 sensor (I2C)

## Software
- C/C++
- FreeRTOS
- Arduino IDE
- Wi-Fi

## Architecture
- Sensor reading task
- Data processing
- Output via serial
